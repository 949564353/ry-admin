## 平台简介

若依是一套全部开源的快速开发平台，毫无保留给个人及企业免费使用。

* 前端采用Vue、Element UI。
* 后端采用Spring Boot、Spring Security、Redis & Jwt。
* 权限认证使用Jwt，支持多终端认证系统。
* 支持加载动态权限菜单，多方式轻松权限控制。

## 改进须知

* 这只是若依3.8.8版本的后台管理部分。
* 会持续对后台部分进行插件的完善和补充。
* 本项目只用来学习和总结，留存记录，方便日后所用。
